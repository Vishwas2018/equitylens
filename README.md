# EquityLens

EquityLens is a deterministic Australian property investment analysis platform. It provides landlords and their advisers with accurate, ATO-aligned cashflow forecasts, capital gains tax modelling, land tax calculations, and scenario analysis — all powered by a pure TypeScript engine with no AI-generated numbers.

See [`docs/process/README.md`](docs/process/README.md) for the operating model, sprint plan, and daily ritual that governs this build.

> **All changes flow through [`docs/process/daily-ritual.md`](docs/process/daily-ritual.md). No direct commits to `main`.**
